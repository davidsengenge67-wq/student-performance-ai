# Student Performance AI Dashboard (Machine Learning + Data Analysis + Streamlit App)

## Project Overview

This project is an end-to-end Machine Learning and Data Science system that analyzes student academic performance using study hours and attendance data.

It includes exploratory data analysis (EDA), data visualization, and a trained regression model integrated into an interactive Streamlit dashboard that allows real-time predictions and insights.

---

## Project Objective

To analyze and understand how student study habits and attendance impact academic performance, and to build a predictive system that estimates student marks using machine learning techniques.

---

## Features

- Exploratory Data Analysis (EDA)
- Data Visualization (Matplotlib & Seaborn)
- Machine Learning Model (Linear Regression)
- Real-time Student Marks Prediction
- Interactive Streamlit Dashboard
- Data Filtering & Export Functionality
- Insight Generation from Data Patterns

---

## Key Insights

- Students with higher study hours tend to achieve better marks
- Attendance has a strong positive correlation with academic performance
- Combined effect of study hours and attendance significantly improves prediction accuracy
