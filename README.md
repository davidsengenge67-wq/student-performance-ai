# Student Performance Analysis & Prediction

This project uses Machine Learning to predict student performance based on study hours and attendance.

## Dataset
Derived from real student records and enhanced using feature engineering.

## Features
- Data cleaning
- Feature engineering
- Linear Regression model

## How to Run

pip install -r requirements.txt  
python model/train_model.py  
python app/app.py
